# Contributing to Hybrid Voice & Vision Automation

Thanks for your interest! Here's how to contribute:

## Getting Started
1. Fork the repository
2. Clone your fork: `git clone <your-fork>`
3. Create a feature branch: `git checkout -b feature/your-feature`

## Development Guidelines
- Keep Python code clean and PEP 8 compliant
- Add comments for complex logic
- Test on Raspberry Pi 4 before submitting
- Include both auto and voice modes in testing

## Submitting Changes
1. Commit with clear messages: `git commit -m "Add feature: description"`
2. Push to your fork: `git push origin feature/your-feature`
3. Open a Pull Request with detailed description
4. Link any related issues

## Reporting Issues
- Use GitHub Issues for bugs
- Include Raspberry Pi model and OS version
- Provide error messages and steps to reproduce

## Code Quality
- Use meaningful variable names
- Comment non-obvious code sections
- Test with different lighting conditions
- Test voice commands with background noise
